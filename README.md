# Pagina web personal - Eros Fabian Caldenhoven Loayza

Este proyecto es una pagina web estatica creada para publicarse en GitHub Pages.

## Archivos

- `index.html`: estructura y contenido de la pagina.
- `styles.css`: diseno visual elegante y responsive.

## Como publicarla en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube `index.html`, `styles.css` y este `README.md`.
3. Entra a `Settings` > `Pages`.
4. En `Build and deployment`, selecciona la rama `main` y la carpeta `/root`.
5. Guarda los cambios y espera a que GitHub genere el link de tu sitio.
